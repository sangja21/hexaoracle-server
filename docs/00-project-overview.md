# 📜 Project Overview — HexaOracle

## 1. 프로젝트 개요
**HexaOracle**은 주역(I-Ching)의 괘(卦)를 디지털로 구현하여, 사용자가 직접 동전 던지기 방식(척전법)을 시뮬레이션하고 AI 해석을 받을 수 있는 **웹 기반 점괘 서비스**입니다.  
프론트엔드에서 사용자가 6효를 완성하는 과정을 진행하면, 백엔드에서 괘 정보를 조회하고 LLM(Language Learning Model) 기반 해석을 제공합니다.

---

## 2. 주요 특징
- **동전 던지기 시뮬레이션**: 사용자가 3개의 동전을 6회 던져 본괘와 지괘를 생성
- **AI 기반 해석**: GPT API와 연동해 괘와 효사(爻辭)에 대한 자연어 해석 제공
- **사용자 맞춤 기록**: 로그인 계정을 통해 과거 점괘 및 해석을 조회 가능
- **웹 환경 지원**: PC/모바일 웹앱 지원
- **구독/결제 기능(향후)**: 프리미엄 해석 및 기록 관리 기능 제공 예정

---

## 3. 기술 스택
- **Frontend**: React
- **Backend**: Spring Boot
- **Database**: MySQL
- **Infra**: Docker
- **AI/LLM**: OpenAI API

---

## 4. 주요 기능
1. **회원가입 및 로그인**
    - SNS 로그인 확장 가능 구조
2. **점괘 생성**
    - 프론트에서 동전 던지기 로직 실행
    - 결과를 백엔드로 전송, 본괘/지괘 판별
3. **AI 해석**
    - DB의 괘/효사 정보와 사용자 질문을 결합
    - LLM에 프롬프트 전송 후 해석 반환
4. **히스토리 관리**
    - 사용자별 Oracle 기록 저장 및 조회
5. **결제/구독 (추후 구현)**

---

## 5. 서비스 대상
- 오리엔탈리즘에 관심 있는 영미권 유저

---

## 6. 서비스 플로우 (요약)
1. 웹사이트 접속
2. (선택) 회원가입 / 로그인
3. 질문 입력
4. 3개의 동전을 6회 던져 괘 생성
5. AI 해석 제공
6. 히스토리 저장 및 조회

---

## 7. 향후 확장 아이디어
- 64괘 카드 콜렉션 기능
- 주역 관련 유튜브 채널 개설 및 서비스 홍보
- 결제 방식 및 구독 모델 정교화
